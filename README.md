# How to buid

* Install [Node.js](https://nodejs.org)

* Clone this repo

  ```bash
  git clone https://github.com/themarketwizard/tradingview-desktop.git
  ```

* Enter folder

  ```bash
  cd tradingview-desktop
  ```

* Download electron

  ```bash
  npm install
  ```

* Build

  ```bash
  npm run build
  ```
